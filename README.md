# crc-organoid-scrnaseq

Bachelor thesis project focused on the application of single-cell RNA-seq analysis
to improve the efficiency of colorectal cancer organoid cultures.

Project overview
This repository contains an exploratory scRNA-seq analysis workflow applied to
colorectal cancer organoid data.  
The goal of the project is to characterize cellular composition and signaling
features of organoid cultures using transcriptomic data.

The project is conducted as part of a Bachelor thesis in bioinformatics.

Dataset
- scRNA-seq dataset: GSE178341
- Data are not included in this repository
- Publicly available via GEO
